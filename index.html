JMAX

A problem solver web app that allows users to post problems they meet in daily life and look for help from other users. They can create requests under three categories, Home Repair, Delivery and Driver. Other users can respond to the requests by providing offers to the requesters. Users can chat with each other privately to have a conversation online and requesters can accept the offer and add comments to the offers which act like reviews.

Yep!

A web app clone of Yelp that allows users to view restaurants with exact locations on google map. Users can search for restaurants they are interested in and displays basic info of the specific restaurant they click on. Users can filter the restaurants based on price range, zip code and ratings.

Hey Wake Up!

A browser game built with JavaScript that challenges the users' clicking speed on dog images within a limited time duration. This game took insipration from daily lectures where students trying hard to stay awake when the instructors are not looking at them. Users are expected to keep clicking on each dog image to make sure their energy level don't drop below 50% when the timer stops. When it's below 50%, it means the dogs are falling asleep while above 50% means the dogs are energetic.
